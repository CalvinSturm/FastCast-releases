# GitHub Pages setup

This repository can serve the static landing page from the repository root.

To enable GitHub Pages:

1. Open the repository on GitHub.
2. Go to Settings -> Pages.
3. Set Source to "Deploy from a branch".
4. Set Branch to `main`.
5. Set Folder to `/root`.
6. Save.

After Pages is enabled and the branch is merged, the landing page should be available at:

```text
https://calvinsturm.github.io/FastCast-releases/
```
